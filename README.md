# Plagiarism Detector // Swiss Design
A high-performance text analysis tool built with Python and Streamlit.

## Features
- **TF-IDF Vectorization:** Converts text into numerical vectors for comparison.
- **Cosine Similarity:** Measures the mathematical distance between documents.
- **Swiss Design UI:** Minimalist, high-contrast interface for better UX.

## Technical Stack
- Python, Streamlit, Scikit-learn, Pandas, Regex.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `streamlit run plagr_app_1.py`
